Redis中的集合

Redis 的 Set 是 String 类型的无序集合。集合成员是唯一的，这就意味着集合中不能出现重复的数据。

Redis 中集合是通过哈希表实现的，所以添加，删除，查找的复杂度都是 O(1)。

集合中最大的成员数为 232 - 1 (4294967295, 每个集合可存储40多亿个成员)。

常用的Redis的集合的函数有以下一些：

sadd

用法：sadd key member1 [member2]

作用：向集合添加一个或多个成员

srem

用法：srem key member1 [member2]

作用：移除集合中一个或多个成员

spop

用法：spop key count / spop key

作用：移除并返回集合中的count/一个随机元素

scard

用法：scard key

作用：获取集合的成员数

smembers

用法：smembers key

作用：返回集合中的所有成员

sismember

用法：sismember key member

作用：判断 member 元素是否是集合 key 的成员

sinter

用法：sinter key1 key2 [key3]

作用：返回给定所有集合的交集

sunion

用法：sunion key1 key2 [key3]

作用：返回给定所有集合的并集

sdiff

用法：sdiff key1 key2 [key3]

作用：返回给定所有集合的差集
