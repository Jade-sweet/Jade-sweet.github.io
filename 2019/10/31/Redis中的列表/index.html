Redis中的列表

Redis列表是简单的字符串列表，按照插入顺序排序。你可以添加一个元素到列表的头部（左边）或者尾部（右边）

一个列表最多可以包含 232 - 1 个元素 (4294967295, 每个列表超过40亿个元素)。

Redis列表常用的函数有以下一些：

lpush
方法：lpush key value1 [value2]
作用：将一个或多个值插入到列表头部

rpush
方法：rpush key value1 [value2]
作用：将一个或多个值插入到列表尾部

lpop
方法：lpop key
作用：移出并获取列表的第一个元素，返回值为移除的元素。

rpop
方法：rpop
作用：移出并获取列表的最后一个元素，返回值为移除的元素。

brpop
方法：brpop key timeout
作用：移出并获取列表的最后一个元素， 如果列表没有元素会阻塞列表直到等待超时或发现可弹出元素为止。

blpop
方法：blpop key timeout
作用：移出并获取列表的第一个元素， 如果列表没有元素会阻塞列表直到等待超时或发现可弹出元素为止。

方法跟brpop一致，不过这个是从左边弹出

lrange
方法：lrange key start_index stop_index
作用：返回列表开始下标和结束下标对应的元素

lrem
方法：lrem key count value
作用：移除列表元素,若指定count，则最多移除count个value

lindex
方法：lindex key index
作用：通过索引获取列表中的元素

linsert
方法：linsert key before/after value
作用：在列表的元素前或者后插入元素

lset
方法：lset key index value
作用：通过索引设置列表元素的值
